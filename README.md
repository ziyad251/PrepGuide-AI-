# PrepGuide AI

PrepGuide AI is a web application that helps users prepare for interviews by generating personalized interview questions and a preparation roadmap based on their resume and the job description.

## Features

- User Registration & Login
- Resume Upload (PDF)
- Job Description Analysis
- Personalized Interview Questions
- Preparation Roadmap
- User Profile Management

## Tech Stack

### Frontend
- React
- Vite
- SCSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
